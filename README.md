# Customer Segmentation and Classification Using Machine Learning

## 📌 Deskripsi Proyek

Repositori ini berisi proyek Machine Learning yang mencakup dua pendekatan utama, yaitu klasifikasi dan clustering. Proyek ini dikembangkan untuk menganalisis data menggunakan berbagai teknik machine learning mulai dari tahap preprocessing, eksplorasi data, pelatihan model, evaluasi, hingga penyimpanan model.

Tujuan proyek ini adalah membangun model yang mampu melakukan prediksi dan segmentasi data secara efektif menggunakan algoritma machine learning.

## ▶️ Cara Menjalankan Proyek

Clone repository:


```bash
cd customer-segmentation-and-classification
```

```bash
pip install -r requirements.txt
```

